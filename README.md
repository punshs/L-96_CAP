# Computer-Assisted Proof for Lorenz 96 Model Hypoellipticity

This repository contains verification scripts for the paper:

**"Non-uniqueness of stationary measures for stochastic systems with almost surely invariant manifolds"**
by Jacob Bedrossian, Alex Blumenthal, and Sam Punshon-Smith.

## Overview

The L96 model is forced on every 4th mode ($I = 4\mathbb{Z}/N\mathbb{Z}$, gap 4). These scripts verify two key algebraic properties:

1. **Lie algebra generation (Appendix C):** The Lie algebra generated by $\{M_k : k \in I\}$ equals $\mathfrak{sl}(H_I^\perp)$.
2. **Linear instability (Section 5):** The transverse linearization $DB(y)|_{H_I^\perp}$ admits eigenvalues with positive real part for suitable $y \in H_I$.

## Files

| File | Description |
|---|---|
| `elementary_matrices_proof_gap4.ipynb` | Main CAP notebook. Computes Lie brackets of $M_4, M_8, M_{12}$ up to depth 4 for $N=12$, verifies that elementary matrices $E_{3,1}$, $E_{6,2}$, $E_{7,4}$ are generated. Combined with shift invariance, this proves $\mathfrak{sl}(H_I^\perp)$ generation for all $N = 4K$, $K \geq 3$. |
| `gap3_quotient_bracket.py` | Symbolically verifies the gap-3 case (Remark 4.2). It computes the Lie bracket rank on both the full 6D transverse space (rank 29/35, showing obstruction) and the 5D quotient space (rank 24/24, showing exact successful span). |
| `symbolic_eigen.py` | Symbolically computes the $9 \times 9$ matrix $DB(y)|_{H_I^\perp}$ for $y = a(e_0 + e_4 + e_8)$ and factors its characteristic polynomial, verifying the instability used in Lemma 5.7. |

## Requirements

- Python 3
- `sympy` (exact rational arithmetic)
- `jupyter` (for the notebook)

## Usage

```bash
# Run the CAP notebook
jupyter nbconvert --execute elementary_matrices_proof_gap4.ipynb

# Verify gap-3 full space and quotient space bracket span
python3 gap3_quotient_bracket.py

# Verify characteristic polynomial factorization
python3 symbolic_eigen.py
```
